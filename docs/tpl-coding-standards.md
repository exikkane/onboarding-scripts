# Читать когда: пишешь/меняешь TPL/HTML
# НЕ читать когда: задача только про PHP

## Назначение
Поддерживать валидный, безопасный и расширяемый шаблонный слой.

## HTML/Smarty правила
1. Теги и атрибуты в нижнем регистре.
2. Значения атрибутов в двойных кавычках.
3. Закрывать теги корректно.
4. Не использовать inline-CSS.
5. Использовать Smarty comments `{* ... *}`.

## Include-параметры
Для булевых флагов использовать `true/false`, не `"Y"/"N"`.

## Hooks и overrides
- Предпочитать `hooks/*.pre.tpl` и `hooks/*.post.tpl`.
- `overrides` использовать только если pre/post не покрывают задачу.

## Процесс
- После изменений в TPL выполнять cache clear.
